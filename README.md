# 🦜 Duolingo Max + Autolingo

A free browser extension that merges two powerful Duolingo tools into one:

- 🏆 **Duolingo Max Features** — Unlocks Max/Super subscription features for free, including unlimited hearts, AI explanations, mistake reviews, and more.
- ⚡ **Autolingo** — Auto-completes Duolingo lessons for you. Click the overlay button on any lesson bubble to auto-solve it, or use the Solve/Solve & Skip buttons manually inside a lesson.

---

## ✨ Features
- Unlimited hearts — never lose a life mid-lesson
- Duolingo Max/Super subscription unlocked for free
- Auto-complete lessons with one click from the lesson map
- Manual Solve & Solve+Skip buttons in the popup
- Solve delay control (ms) for more natural pacing
- Keyboard shortcuts: `Ctrl+Enter` (solve & skip), `Alt+Enter` (solve)
- Multiple patch modes — if one breaks after a Duolingo update, switch to another
- Auto-sync patch — automatically uses the best working patch remotely

## 🚀 Installation

### Chrome / Edge / Brave / Opera
1. Download and extract `duolingo_max_autolingo.zip`
2. Open your browser and go to `chrome://extensions` (or `edge://extensions`)
3. Enable **Developer Mode** (top right toggle)
4. Click **Load unpacked** and select the extracted folder
5. Open [Duolingo](https://www.duolingo.com) and enjoy!

### Firefox
1. Download `duolingo_max_autolingo_firefox.xpi`
2. Open Firefox and go to `about:addons`
3. Click the **⚙️ gear icon** → **Install Add-on From File...**
4. Select the downloaded `.xpi` file
5. Click **Add** when prompted
6. Open [Duolingo](https://www.duolingo.com) and enjoy!

> **Note:** Firefox may show a warning since the extension is not signed by Mozilla. This is normal for self-distributed extensions. To bypass: go to `about:config` → set `xpinstall.signatures.required` to `false`.

## ⚙️ Usage
- Click the extension icon to open the popup
- **🏆 Max Features tab** — access patch options via the ⚙️ Options button
- **⚡ Autolingo tab** — toggle auto-solve, set delay, or use manual solve buttons
- On the lesson map, click the **overlay button** on any lesson bubble to auto-complete that specific lesson
- Opening a lesson the normal way keeps it in manual mode

---

## 🔧 Patch Modes (Options Page)
| Category | What it unlocks |
|----------|----------------|
| Max Features | Gold subscription, AI features |
| Super Features | Super subscription, no ads |
| Unlimited Hearts | Removes heart limit |

If one patch stops working after a Duolingo update, open Options and try a different patch number.
